# CEDEN Explorer

Interactive data exploration tool for the California Environmental Data Exchange Network (CEDEN) dataset.

## Overview

CEDEN Explorer allows efficient analysis of a 5GB environmental health dataset. The app provides:

- Map-based selection of monitoring stations  
- Summary statistics (mean, min, max) for quick exploratory data analysis  
- CSV export for further analysis  
- Easy identification of regions with elevated analytes, such as mercury  

This tool was developed to improve team productivity and replace manual query workflows.

**Technologies:** Python, FastAPI, BigQuery, Google Cloud Storage, JavaScript, Docker  
**Live App:** https://ceden-app-717905783428.us-central1.run.app/
